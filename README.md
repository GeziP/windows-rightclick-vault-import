# windows-rightclick-vault-import

Windows right-click vault import tool planning repository.

## Development

Install the Rust toolchain, then run commands from `kbintake/`:

```powershell
cargo build
cargo run -- doctor
cargo run -- import <path>
cargo run -- agent
cargo run -- jobs list
```
